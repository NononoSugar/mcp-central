# How-to-use

## Installation

```shell
pip install -r requirements.txt
crawl4ai-setup
crawl4ai-doctor
```

## Dev

```shell
fastmcp dev mcp.py
```

## config

```json
{
    "crawl4ai": {
      "command": "fastmcp",
      "args": [
        "run",
        "path/to/crawl4ai/mcp.py"
      ]
    }
}
```
